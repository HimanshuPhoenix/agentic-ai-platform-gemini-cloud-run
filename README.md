# Adaptive AI Agent Platform

An extensible, production-ready AI agent system built using modern Generative AI concepts.

This project demonstrates how to design, build, and deploy an **autonomous AI agent** capable of reasoning, interacting with real-world tools, and delivering structured insights.

---

## Project Overview

The Adaptive AI Agent Platform is designed to:

- Accept user queries
- Perform reasoning using an AI agent
- Fetch real-world data via external tools (APIs)
- Generate structured, meaningful responses
- Be deployed as a scalable cloud service (Google Cloud Run)

---

## Key Features

- 🤖 **Agentic AI System**
  - Goal-oriented reasoning and response generation

- 🔌 **Tool Integration (MCP Concept)**
  - Connects to external APIs (e.g., Wikipedia)

- ☁️ **Cloud Deployment**
  - Fully deployable on Google Cloud Run

- 🧩 **Modular Architecture**
  - Easily extendable for new tools, domains, and use cases

- 📊 **Structured Responses**
  - Outputs organized insights instead of raw text

---

## Architecture
 - User Input → FastAPI → AI Agent → Tool (API) → Response

## Getting Started (Local Setup)

### 1. Clone the repository

```bash
git clone https://github.com/HimanshuPhoenix/agentic-ai-platform-gemini-cloud-run.git
cd adaptive-ai-agent-platform-gemini-cloud-run
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the application
```bash
uvicorn app.main:app --reload
```

### 4. Test the API

Open:

  http://127.0.0.1:8000/docs

Use /query endpoint to interact with the agent.

---

## Deployment (Google Cloud Run)

This project is containerized and can be deployed using:
```bash
gcloud builds submit --tag gcr.io/YOUR_PROJECT_ID/ai-agent
gcloud run deploy ai-agent --image gcr.io/YOUR_PROJECT_ID/ai-agent --platform managed --allow-unauthenticated
```
🧪 Example Query
POST /query

Input:
"Summarize recent research in AI agents"

Output:
Structured insights fetched and generated by the agent.
